# AI Notebooks

This repository collects a series of educational Jupyter notebooks covering modern topics in Artificial Intelligence and Deep Learning.

Its goal is to provide concise, self-contained introductions to cutting-edge AI methods through intuitive explanations and minimal working examples.

## Available notebooks

### Conditional Flow Matching



The first notebook in this series introduces **Conditional Flow Matching (CFM)** and includes three minimal implementations:

* **Unconditional generation**
* **Conditioned generation** 
* **Discrete Flow Matching**

Each example is designed to highlight the core ideas of the method while keeping the implementation as simple and accessible as possible.
